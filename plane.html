Плоскость

Общее уравнение плоскости

Общее уравнение плоскости в пространстве: $Ax + By + Cz + D = 0$

Уравнение плоскости, проходящей через точку перпендикулярно вектору нормали

Уравнение плоскости, проходящей через точку $M_0(x_0, y_0, z_0)$ и имеющей вектор нормали $\vec{n}(A, B, C)$: $$ A(x - x_0) + B(y - y_0) + C(z - z_0) = 0 $$

Уравнение плоскости в отрезках

Уравнение плоскости в отрезках, отсекаемых на осях $Ox, Oy, Oz$: $\frac{x}{a} + \frac{y}{b} + \frac{z}{c} = 1$

Параметрическое уравнение плоскости

Параметрическое уравнение плоскости, проходящей через точку $M_0(x_0, y_0, z_0)$ с направляющими векторами $\vec{a}(l_1, m_1, n_1)$ и $\vec{b}(l_2, m_2, n_2)$: $$ \begin{cases} x = x_0 + l_1u + l_2v \\ y = y_0 + m_1u + m_2v \\ z = z_0 + n_1u + n_2v \end{cases} $$ где $u, v$ — параметры.

Угол между двумя плоскостями

Угол $\phi$ между двумя плоскостями $A_1x + B_1y + C_1z + D_1 = 0$ и $A_2x + B_2y + C_2z + D_2 = 0$: $$ \cos \phi = \frac{|A_1A_2 + B_1B_2 + C_1C_2|}{\sqrt{A_1^2 + B_1^2 + C_1^2} \sqrt{A_2^2 + B_2^2 + C_2^2}} $$

Расстояние от точки до плоскости

Расстояние $d$ от точки $M_0(x_0, y_0, z_0)$ до плоскости $Ax + By + Cz + D = 0$: $$ d = \frac{|Ax_0 + By_0 + Cz_0 + D|}{\sqrt{A^2 + B^2 + C^2}} $$

Условия параллельности и перпендикулярности плоскостей

Две плоскости $A_1x + B_1y + C_1z + D_1 = 0$ и $A_2x + B_2y + C_2z + D_2 = 0$:

  • Параллельны, если $\frac{A_1}{A_2} = \frac{B_1}{B_2} = \frac{C_1}{C_2}$.
  • Перпендикулярны, если $A_1A_2 + B_1B_2 + C_1C_2 = 0$.
