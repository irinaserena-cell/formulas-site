Векторные пространства

Векторное пространство $V$ над полем $F$ (например, $\mathbb{R}$ или $\mathbb{C}$) — это непустое множество, элементы которого называются векторами, с двумя операциями:

  • **Сложение векторов:** для любых $\mathbf{u}, \mathbf{v} \in V$, $\mathbf{u} + \mathbf{v} \in V$.
  • **Умножение вектора на скаляр:** для любого $\mathbf{u} \in V$ и любого $c \in F$, $c\mathbf{u} \in V$.

Эти операции должны удовлетворять следующим 10 аксиомам:

Аксиомы сложения векторов:

  1. **Коммутативность сложения:** $\mathbf{u} + \mathbf{v} = \mathbf{v} + \mathbf{u}$ для всех $\mathbf{u}, \mathbf{v} \in V$.
  2. **Ассоциативность сложения:** $(\mathbf{u} + \mathbf{v}) + \mathbf{w} = \mathbf{u} + (\mathbf{v} + \mathbf{w})$ для всех $\mathbf{u}, \mathbf{v}, \mathbf{w} \in V$.
  3. **Существование нулевого вектора:** Существует элемент $\mathbf{0} \in V$, называемый нулевым вектором, такой что $\mathbf{u} + \mathbf{0} = \mathbf{u}$ для всех $\mathbf{u} \in V$.
  4. **Существование противоположного вектора:** Для каждого $\mathbf{u} \in V$ существует элемент $-\mathbf{u} \in V$, называемый противоположным вектором, такой что $\mathbf{u} + (-\mathbf{u}) = \mathbf{0}$.

Аксиомы умножения на скаляр:

  1. **Ассоциативность умножения на скаляр:** $c(d\mathbf{u}) = (cd)\mathbf{u}$ для всех скаляров $c, d \in F$ и всех $\mathbf{u} \in V$.
  2. **Единичная скалярная единица:** $1\mathbf{u} = \mathbf{u}$ для всех $\mathbf{u} \in V$, где $1$ — единичный элемент поля $F$.

Аксиомы дистрибутивности:

  1. **Дистрибутивность скаляра по отношению к сложению векторов:** $c(\mathbf{u} + \mathbf{v}) = c\mathbf{u} + c\mathbf{v}$ для всех скаляров $c \in F$ и всех $\mathbf{u}, \mathbf{v} \in V$.
  2. **Дистрибутивность вектора по отношению к сложению скаляров:** $(c + d)\mathbf{u} = c\mathbf{u} + d\mathbf{u}$ для всех скаляров $c, d \in F$ и всех $\mathbf{u} \in V$.

Дополнительные аксиомы (часто включаются в определение поля):

Хотя эти аксиомы относятся к свойствам поля скаляров $F$, они подразумеваются при работе с векторными пространствами:

  1. **Замкнутость сложения:** Если $c, d \in F$, то $c+d \in F$.
  2. **Замкнутость умножения:** Если $c, d \in F$, то $cd \in F$.

Любое множество, удовлетворяющее этим аксиомам, является векторным пространством.
